# Nebula Media Pricing

Nebula Media is the media pillar of Parad0x Labs.

## Open Source (Free, Forever)

Nebula Media is **MIT-licensed and free to self-host**. Clone the repo, run the
pipelines, generate and verify receipts — no license, no lock-in.

This includes:
- Media encoding pipelines (video + image/AVIF live, including the .null page
  mode; audio is on the roadmap)
- Public benchmark documentation and reproducible proof-pack
- Verification tooling and methodology

## Managed Service (Planned)

A hosted service is **planned, not yet live**. When it ships it will add convenience
on top of the open core — never close it. Expected scope:
- Hosted batch encoding and an API
- Automatic on-chain receipt anchoring (via dna-x402)
- Scale, SLAs, and priority support
- Dedicated / private-cloud / air-gapped deployments
- Compliance and audit support

For enterprise inquiries, contact Parad0x Labs via email (**hello@parad0xlabs.com**) or DM on X (**[@Parad0x_Labs](https://x.com/Parad0x_Labs)**).

## Notes

- **Offline Verification:** All encoding and quality verification run locally — no data has to leave your infrastructure.
- **Content Dependency:** Benchmark results are content-dependent and provided for transparency, not as universal guarantees.
- **Open Core:** The full pipeline is MIT-licensed and published here. The managed service adds hosting and scale, not closed code.
